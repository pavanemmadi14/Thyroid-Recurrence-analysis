**🧬 Thyroid Cancer Recurrence Analysis Dashboard**

📌 Project Description

This project presents an interactive healthcare analytics dashboard designed to analyze thyroid cancer recurrence patterns using patient clinical data. The dashboard helps identify key risk factors influencing recurrence and survival outcomes through visual analytics and predictive insights.

The solution was developed using Power BI and focuses on understanding the relationship between tumor characteristics, patient demographics, treatment types, and recurrence probability.

**📊 Project Overview**

Thyroid cancer recurrence is influenced by multiple clinical and demographic factors. This project analyzes patient data to identify patterns and trends associated with recurrence risk.

The dashboard provides:

Executive summary of patient statistics

Recurrence and survival rate analysis

Tumor size and stage impact evaluation

Clinical risk factor insights

Interactive filters for exploratory analysis

Predictive recurrence risk visualization

🎯 Project Goal

The goal of this project is to demonstrate how data analytics and visualization techniques can support healthcare insights and assist in clinical decision analysis.

📊 Key Insights

Some insights identified from the analysis include:

Recurrence risk varies across different cancer stages.

Tumor size shows correlation with recurrence probability.

Certain age groups demonstrate higher recurrence patterns.

Presence of adenopathy increases recurrence risk.

Survival trends fluctuate across follow-up years.
